docker-compose install for window
----------------------------------------------------

$ docker-machine ssh

sudo chown -R $USER:$USER /usr/local/bin/

curl -L https://github.com/docker/compose/releases/download/1.25.4-rc2/docker-compose-Linux-x86_64 -o /usr/local/bin/docker-compose

chmod +x /usr/local/bin/docker-compose

------------------------------------------------------
